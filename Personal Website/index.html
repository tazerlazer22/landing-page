Projects

Stock Price Prediction

Harnessing the power of deep learning, I developed a cutting-edge predictive model using Long Short-Term Memory (LSTM) networks, achieving an impressive 96% accuracy in forecasting stock prices with data from Yahoo Finance. This innovative project leverages advanced machine learning techniques to provide highly accurate predictions, demonstrating the potential of AI in revolutionizing financial analysis and investment strategies.

Banknote Authentication

In this exciting project, I leveraged advanced machine learning techniques to develop a highly accurate banknote authentication system. Utilizing the power of Random Forest Classifiers, I meticulously tuned the model parameters through an extensive GridSearchCV process to identify the optimal settings. With features extracted from real-world data, the final model achieved an exceptional accuracy rate, demonstrating its efficacy in distinguishing between genuine and counterfeit banknotes.

Investment Portfolio Analysis

In this project, I developed an interactive investment portfolio dashboard using Streamlit and yFinance. Users can input their chosen assets and corresponding weights, and the application fetches historical stock data to analyze portfolio performance and risk. The dashboard compares the portfolio's cumulative returns and risk against the S&P 500 benchmark, providing visualizations and insights into portfolio composition and volatility. This tool helps investors assess and understand the performance and risk profile of their investments in a clear and user-friendly interface.
